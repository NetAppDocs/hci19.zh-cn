= NetApp HCI 配置和管理文档
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
NetApp HCI 可提供存储和计算资源，并将其组合在一起，以构建一个以 NetApp Element 软件功能为后盾的 VMware vSphere 环境。

您可以使用 NetApp 混合云控制界面升级，扩展和监控系统，并使用适用于 vCenter Server 的 NetApp Element 插件管理 NetApp HCI 资源。
